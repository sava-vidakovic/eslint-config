### Install

```
npm install git+ssh://git@gitlab.com/sava.vidakovic/eslint-config-fww.git
```

### Install dependencies

```
npm install @typescript-eslint/parser typescript prettier
```

### Prettier Config

Create `.prettierrc` file on the root

```
{
    "singleQuote": true,
    "tabWidth": 2,
    "printWidth": 100,
    "trailingComma": "es5",
    "arrowParens": "avoid"
}
```
